### Event Types
The following outbound event types are currently in a Beta status.  A Webhook Subscription can be created with these event types for experimentation.  

#### service.created
`data.type` is [`service_reference`](#service_reference)

Sent when a Service is created.

#### service.updated
`data.type` is [`service_reference`](#service_reference)

Sent when a Service is updated.

#### service.deleted
`data.type` is [`service_reference`](#service_reference)

Sent when a Service is deleted.

### Event Data Types

#### service_reference

```json
{
    "html_url": "https://acme.pagerduty.com/services/PF9KMXH",
    "id": "PF9KMXH",
    "self": "https://api.pagerduty.com/services/PF9KMXH",
    "summary": "testing service updates",
    "type": "service_reference"
  }
```
